# MU* Code Repository
### Compiled and Maintained by Daemon

A comprehensive collection of code files, scripts, and full codebases for **Multi-User Experience (MUX)** and **Multi-User Shared Hallucination (MUSH)** environments. This repository serves as a centralized archive for server-side softcode, system scripts, and various bot implementations.

---

## üìÇ Repository Structure

The repository is organized alphabetically and by category to facilitate easy navigation:

### Special Categories
* **`!Codebases/`**: Complete server source code and foundational systems.
* **`!MUX Files/`**: Specific scripts and configurations tailored for TinyMUX environments.
* **`!PennMUSH/`**: Code specialized for the PennMUSH flavor of servers.
* **`!Wing Commander MUSH/`**: Specialized code and assets related to the Wing Commander themed MUSH.
* **`$bots/` & `.bots/`**: Automated bot scripts and interaction systems.
* **`$Misc/`**: General utilities, snippets, and tools that do not fit into specific categories.

### Alphabetical Archives (A-Z)
Softcode snippets, functions, and objects organized by their primary name or function. This includes:
* **Global Systems**: Economy, Weather, Space, and Combat.
* **Player Tools**: Chargers, descriptions, and inventory systems.
* **Admin Utilities**: Security logs, quota management, and building tools.
* **Communication**: Arrays, radio systems, and paging enhancements.

## üöÄ Usage

Most files in this repository are intended to be "quoted" or uploaded into a running MU* server.

1.  **Navigate** to the directory containing the system you need.
2.  **Review** the file content for any specific configuration settings (e.g., `#define` constants or attribute settings).
3.  **Upload**: Use your preferred MU* client (such as Potato, TinyFugue, or MUSHclient) to quote the text into the game. 
    * *Note: Ensure you have the appropriate permissions (usually Wizard or Royalty) if the code requires setting restricted flags.*
